# 1% Daily — Habit Tracker

A mobile-first PWA habit tracker. Get 1% better every day.

## Phase 1 (Current)
- Single index.html, Vanilla JS, Tailwind CSS
- localStorage for data persistence
- PWA installable via manifest.json

## Phase 2 (Planned)
- Vite + Vanilla JS (multi-file)
- Supabase for auth and database
- Google OAuth
- Service Worker for offline support

## Deploy
Hosted on Netlify free tier.
